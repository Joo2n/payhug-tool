서명·도장 사진 한 장의 배경을 지워 투명 PNG로 만드는 도구 · 운영팀용 · 2026.07
서명·도장 사진 한 장을 빠르게 처리할 때 쓰는 도구예요. 슬라이더로 결과를 직접 다듬을 수 있는 게 특징입니다. 설치 없이 크롬에서 바로 쓰면 되고, 파일이 외부로 나가지 않아요.
여러 장·PDF를 한꺼번에 처리할 땐 여러 파일 배경 제거(별도 가이드)를 쓰세요.

1도구를 열고, 사진을 끌어다 놓으세요

서명·도장이 찍힌 사진/스캔/PDF 한 개를 넣으면 됩니다. 다른 파일로 바꾸려면 [다른 파일 열기]를 누르세요.

2결과를 보면서 슬라이더로 다듬으세요

넣자마자 배경이 지워진 결과가 보여요. 강도는 오른쪽으로 갈수록 옅은 잉크·도장 질감까지 살아나고, 가장자리 부드러움은 오른쪽으로 갈수록 테두리가 매끄러워집니다. 도장은 보통 둘 다 100이 제일 잘 나와요.

3필요한 부분만 확대해서 드래그하세요

[+] 버튼이나 Ctrl+휠(트랙패드 핀치)로 확대하고, 마우스로 원하는 부분을 드래그하면 그 영역만 잘라서 저장돼요. 드래그를 안 하면 전체가 저장됩니다.

4PNG 다운로드 — 이렇게 나옵니다

어두운 배경에 얹어도 테두리에 흰 잡티 없이 깨끗해요. 파일 이름은 원본명(누끼).png로 저장됩니다.

예: 도장.jpg → 도장(누끼).png · 계약서.pdf 3페이지 → 계약서 p3(누끼).png

자주 물어볼 것 같은 것

Q. 인터넷이 필요한가요? 이미지는 인터넷 없이도 됩니다. PDF를 넣을 때만 인터넷이 필요해요. (파일 자체는 밖으로 안 나가요)
Q. 여러 장을 한꺼번에 하려면요? 이 도구는 한 장씩이에요. 여러 장·PDF 일괄 처리는 여러 파일 배경 제거를 쓰세요 (파일 여러 개 드래그 + 전체 저장).
Q. 어두운 배경이나 손이 같이 찍힌 사진도 되나요? 이 도구는 "밝은 종이 + 어두운 잉크" 전용이에요. 배경이 복잡한 사진은 잘 안 될 수 있으니, 가급적 종이만 나오게 찍어주세요.
