🔤 한글 파일명 고치기 — 사용 가이드

맥에서 만든 파일의 깨진 한글 이름(자소분리)을 고쳐 주는 도구 · 운영팀용 · 2026.07
맥에서 만든 파일을 Jira·윈도우로 보내면 한글이 ㅎㅏㄴㄱㅡㄹ처럼 조각조각 깨져 보일 때가 있어요. 이 도구에 넣으면 어떤 이름이 깨질지 미리 보여주고, 안 깨지는 이름으로 바꿔서 돌려줍니다. 파일 내용은 전혀 건드리지 않고 이름만 고쳐요. 설치 없이 크롬에서 바로 쓰면 되고, 파일이 외부로 나가지 않아요.

1왜 깨질까요? — 맥은 한글을 "조각"으로 저장해요

같은 한글.txt라도, 맥은 파일 이름을 ㅎ+ㅏ+ㄴ+ㄱ+ㅡ+ㄹ처럼 자음·모음 조각으로 풀어서 저장해요. 맥 안에서는 알아서 합쳐 보여주니까 티가 안 나는데, 윈도우·Jira는 완성된 글자를 기대하기 때문에 조각을 그대로 보여줘서 ㅎㅏㄴㄱㅡㄹ.txt처럼 깨져 보이는 거예요. 이걸 자소분리라고 부릅니다.

파일이 고장난 게 아니라 이름 표기 방식만 다른 것이라, 이름을 "완성된 글자" 방식으로 다시 써 주면 끝나요. 이 도구가 그 일을 해 줍니다.

2도구를 열고, 파일·폴더·zip을 끌어다 놓으세요

파일 여러 개, 폴더 통째, .zip 모두 알아서 판별해요. 폴더는 [폴더 선택] 버튼으로 고를 수도 있고, 복사해 둔 파일은 Cmd+V 붙여넣기로도 넣을 수 있어요. zip을 넣으면 압축을 풀지 않고 내부 파일명까지 들여다봅니다.

3진단 표에서 어떤 이름이 깨지는지 확인하세요

넣자마자 이름을 하나씩 검사해서 표로 보여줘요.
· 빨간 하이라이트 + "자소분리됨" — 윈도우·Jira에서 깨져 보일 이름이에요. 오른쪽에 고친 이름이 미리 보여요.
· "정상" — 그대로 보내도 안 깨지는 이름이에요. 전부 정상이면 "이미 정상이에요" 안내가 떠요.
· zip은 제목 줄에 파일명 인코딩도 표시해요 — "UTF-8(맥 표준)", "EUC-KR(윈도우 구형)"처럼 어떤 방식으로 저장됐는지 알려줍니다.

맥에서 압축한 zip에 몰래 들어가는 시스템 파일(__MACOSX 폴더, ._로 시작하는 파일, .DS_Store)은 자동으로 걸러져요 — 목록에 나오지 않고, 변환된 결과 zip에서도 빠집니다. 몇 개를 뺐는지 표 위에 안내가 뜨고, 눌러서 어떤 파일이었는지 확인할 수 있어요. 이 파일들은 윈도우·Jira에서 쓰레기 파일로만 보이니 빠지는 게 정상이에요.

4저장 버튼을 눌러 변환하세요

넣은 것에 따라 버튼이 알맞게 나타나요.
· 파일 1개 → [고친 이름으로 저장] 한 번이면 끝.
· 파일 여러 개 → [각각 저장]으로 낱개로 받거나, [zip으로 묶어 저장]으로 한 번에 받아요.
· 폴더 → 폴더 구조를 그대로 유지한 zip으로 받아져요.
· zip → [고친 zip 저장]을 누르면 압축을 풀 필요 없이 내부 파일명만 고친 새 zip이 내려와요. 파일 내용은 그대로예요.

[각각 저장]처럼 여러 개를 받을 때 브라우저가 "여러 파일 다운로드 허용"을 물어보면 [허용]을 눌러 주세요. zip으로 묶을 때 [용량 줄이기]를 켜면 파일이 작아지는 대신 조금 느려져요.

5Jira에 올릴 땐, 받은 파일을 그대로 첨부하세요

여기서 받은 파일은 이름이 이미 고쳐져 있으니 그대로 Jira에 첨부하면 윈도우 동료에게도 한글이 제대로 보여요. 폴더를 넣었다면 zip으로 받아지는데, zip째 첨부하면 됩니다. 원본 파일 말고 꼭 이 도구에서 받은 쪽을 올리는 것만 기억하세요.

자주 물어볼 것 같은 것

Q. 암호 걸린 zip도 되나요? 안 돼요. 암호 zip은 내부를 다시 쓸 수 없어서 처리할 수 없어요. 압축을 푼 다음 폴더째 넣어 주세요 — 그러면 고친 zip으로 다시 받을 수 있어요.
Q. 폴더를 넣었는데 빈 폴더가 사라졌어요. 빈 폴더는 zip에 포함되지 않아요 (화면에도 같은 안내가 떠요). 파일이 든 폴더만 결과 zip에 들어갑니다.
Q. 반디네이머 같은 앱과 뭐가 다른가요? 설치 없이 크롬에서 바로 쓰는 것에 더해, 폴더 통째·zip 내부 파일명까지 한 번에 처리해요. zip은 압축을 풀지 않고도 내부 이름만 고쳐서 새 zip으로 돌려주고, 어떤 인코딩으로 저장된 zip인지도 알려줍니다.
Q. 인터넷이 필요한가요? 파일이 밖으로 나가나요? 모든 처리는 브라우저 안에서만 이뤄지고 파일은 밖으로 나가지 않아요. 다만 zip을 만들거나 다시 쓸 때 압축 도구(외부 라이브러리)를 불러오는 데만 인터넷이 필요해요.
Q. 파일 내용도 바뀌나요? 아니요. 이름만 바뀌고 내용물은 1바이트도 건드리지 않아요. 이름이 겹치면 이름 2처럼 번호를 붙여 구분해 줍니다.
